## POE

### Fan

Edit `/boot/config.txt` & reboot

40000 means 40°C
50000 means 50°C
65000 means 65°C
etc.

```
# POE Fan Speeds
dtparam=poe_fan_temp0=40000
dtparam=poe_fan_temp1=50000
dtparam=poe_fan_temp2=65000
dtparam=poe_fan_temp3=80000
```
